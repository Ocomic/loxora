# Loxora

> Projects should never lose their memory.

Loxora is an experimental, local-first and model-independent project knowledge and context layer.

**Status:** Controlled planning and preparation
**Working title:** Loxora

This repository currently contains foundational RFCs and proposed Hackathon planning. It does not contain product implementation. Proposed architecture and MVP documents require review before setup or implementation begins.

## Important

Loxora is documentation-driven. Architecture decisions should be documented before implementation whenever reasonably possible.

The proposed Hackathon documents do not authorize scaffolding, dependency installation, database design, or runtime implementation. Approval gates are recorded in RFC-007 and the pre-Hackathon boundary checklist.
